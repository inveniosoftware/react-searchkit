ResultsMultiLayout

ResultsMultiLayout is an uncontrolled component that listens to the application's layout state and renders results in a list (ResultsList) or grid (ResultsGrid) respectively. Can be used in combination with the LayoutSwitcher component to change the results layout in controllable way. By default it renders results as a list.
